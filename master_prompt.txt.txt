# HAMSAFAR — PROJECT BRIEF FOR CLAUDE CODE

You are building HamSafar, a community carpooling web app for Pakistan, as a Human-Computer Interaction (HCI) university project. This is a high-fidelity, mobile-first interactive prototype with mocked data and flows. No real backend. Treat this as a polished, demoable product that a grader will click through on a phone or laptop.

## TECH STACK (USE EXACTLY THIS)

- Next.js 14 with App Router and TypeScript
- Tailwind CSS for styling
- shadcn/ui for components (initialize with "new-york" style, slate base color)
- Lucide React for icons
- Framer Motion for transitions and animations
- React Hook Form + Zod for any form validation
- Zustand for global state (user, auth, rides, trusted contacts)
- localStorage for persistence across sessions
- date-fns for date formatting
- Static placeholder map images (no real Google Maps API key) — use SVG-styled mock maps or a free static map image. For tracking, animate a marker along an SVG path.

## DESIGN PHILOSOPHY

- Mobile-first. Primary canvas is 390px wide.
- On desktop, center the 390px frame inside a styled phone bezel/mockup for demo purposes. Use a subtle device-frame styling with rounded corners, slight shadow, and a neutral background gradient outside the frame.
- Trust-first visual language. The brand emphasizes verified peers, safety, and transparency.
- Clean, modern, restrained. Not playful or childlike. Closer to Careem or Uber in maturity, but warmer and more local.
- Cultural awareness for Pakistan. Use names like Ahmed, Fatima, Hassan, Aisha. Show prices in PKR (₨). Reference real Pakistani cities, neighborhoods, and universities.

### Typography

- Use Inter from Google Fonts as the primary font, loaded via next/font
- Display headings: Inter 700, tight letter-spacing
- Body: Inter 400-500
- Sizes: text-xs (12), text-sm (14), text-base (16), text-lg (18), text-xl (20), text-2xl (24), text-3xl (30)
- Line height: relaxed for body, tight for headings

### Spacing & Layout

- Standard padding inside cards: 16px (p-4)
- Section spacing: 24px (gap-6)
- Page padding: 20px horizontal (px-5)
- Border radius: rounded-2xl (16px) for cards, rounded-xl (12px) for inputs, rounded-full for pills and CTAs
- Shadows: soft, low-elevation. Avoid harsh drop shadows.

### Components Patterns

- Buttons: full-width pill-shaped primary CTA at the bottom of screens (sticky on scroll), height 56px, teal background, white text, semibold
- Secondary buttons: outlined, teal border, teal text
- Verification badges: small pill with checkmark icon — Blue for CNIC, Teal for University, Indigo for Workplace
- Star ratings: gold/amber colored, with numeric next to it
- Bottom navigation: 5 icons (Home, Find, Active Ride, Inbox, Profile), 64px height, fixed bottom, with active state in primary teal

### Iconography

- Use Lucide React icons throughout
- Icon size: 20px default, 24px for navigation, 16px inline with text
- Stroke width: 2
